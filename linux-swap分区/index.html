Linux swap分区配置

创建swap文件

创建swap文件夹

1
2
3

cd /usr
mkdir swap
cd swap/

创建swapfile文件

1

dd if=/dev/zero of=/usr/swap/swapfile bs=1M count=4096

查看swap文件

1

du -sh /usr/swap/swapfile

设置swap分区

将目标设置为swap分区文件

1
2

mkswap /usr/swap/swapfile
swapon /usr/swap/swapfile

查看现在的内存

1

free -m

设置开机自动启用虚拟内存

在etc/fstab文件中加入命令

1

vim /etc/fstab

末尾添加

1

/usr/swap/swapfile swap swap defaults 0 0

重启服务器

1

reboot

重启之后查看现在的内存

1

free -m
