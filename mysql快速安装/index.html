MySql快速安装

Linux安装

安装

1
2
3
4

sudo su
sudo apt update
sudo apt install mysql-server
mysql --version

查看服务

netstat -tap | grep mysql netstat -ano netstat -anp

登录

1

mysql -u root -p

改密码

1
2

ALTER user 'root'@'localhost' IDENTIFIED with mysql_native_password BY '123456';
quit

远程访问授权

1
2

sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf
# bind-address = 127.0.0.1

1
2
3
4
5
6

create user root@'%' identified by '123456';
grant all privileges on *.* to root@'%' with grant option;

use mysql;
update user set host = '%' where user = 'root';
flush privileges;

开关服务

1
2
3
4
5

systemctl restart mysql;
systemctl status mysql;
systemctl start
systemctl stop
systemctl reload

查看

端口（默认3306）

1

SHOW VARIABLES LIKE 'port';

安装位置

1

whereis mysql

配置文件位置

1

/etc/my.cnf

备用用户

1
2
3
4
5

sudo cat /etc/mysql/debian.cnf
# debian-sys-maint
# ztv*************

mysql -u debian-sys-maint -p

Windows源码安装

配置环境变量

1

......\mysql-8.0.31-winx64\bin

编辑配置文件

mysql-8.0.31-winx64目录下写my.ini

1
2
3
4
5

[mysqld]
port=3306
basedir=D:\\dev\\mysql-8.0.31-winx64
datadir=D:\\dev\\mysql-8.0.31-winx64\\data
secure-file-priv=

如果需要跳过登录密码（忘记密码）： my.ini添加：

1

skip-grant-tables

或者：

1

mysqld --console --skip-grant-tables --shared-memory

初始化

1

mysqld --initialize-insecure

创建服务

1
2
3
4
5

mysqld --install mysql8031

mysqld -remove
mysqld -install
mysqld --initialize

开关服务

1
2

net start mysql8031
net stop mysql8031

登录

1

mysql -u root -p 123456

设密码

1
2

flush privileges;
ALTER user 'root'@'localhost' IDENTIFIED BY '123456';
